Command In Bot
====================

.. _installation:

اوامر التشغيل
------------

``تشغيل + شغل + /play``


لتشغيل الاغاني في المجموعات والقنوات

اوامر الادمن
------------

``ايقاف مؤقت + وقف + /pause``
لايقاف الشتغيل مؤقتا

``استكمال + كمل + /rasume``
لتكمله الاغنيه تم ايقافها مؤقتا

``تخطي + /skip``
لتخطي الاغنيه وتشغيل ما بعدها

``انهاء + ايقاف + /end + /stop``
لايقاف التشغيل في المكالمه
